# aula20_bootcamp

Configuração do ambiente
    Criar a pasta do projeto
    Criar o repositório no github com o mesmo nome da pasta
    Fazer o primeiro commit
    Instalar o poetry
        poetry init
        poetry env activate
        touch .gitignore
        pyenv local 3.12.1
        poetry env use 3.12.1
    Estruturar a ideia do projeto em algum mind map e configurar no VS Code
        Criar pasta backend
        Criar pasta frontend
        Criar arquivo docker-compose.yml (raíz do diretório)
        Criar arquivo Dockerfile (dentro de cada pasta)
